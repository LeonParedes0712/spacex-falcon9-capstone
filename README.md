# SpaceX Falcon 9 Capstone Project

This repository contains the final project for the IBM Applied Data Science Capstone.

## Project Objective
The goal of this project is to analyze SpaceX Falcon 9 launch data and predict whether the first stage will successfully land.

## Contents
- Data collection using SpaceX API/static datasets
- Web scraping of Falcon 9 launch records
- Data wrangling and feature engineering
- Exploratory data analysis using SQL and visualizations
- Interactive maps with Folium
- Dashboard with Plotly Dash
- Machine learning classification models

## Final Deliverables
- Final presentation PDF
- Python notebooks/scripts
- Generated datasets
- Screenshots used as evidence
